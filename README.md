# Demon Powder Coating — Demo Website

Demo site built for Kevin Eitniear / Demon Powder Coating, Castle Rock, CO.

**Live URL:** https://stellar-clarity.github.io/demon-powder-coating-website/

## Stack
- Next.js 14 + TypeScript + Tailwind CSS v3
- Static export → GitHub Pages

## Development
```bash
npm install
npm run dev
```

## Deploy
Push to `main` — GitHub Actions handles the rest.
